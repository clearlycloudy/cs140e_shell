pub struct WriteVolatile<T>(_);

A wrapper type that enforces write-only volatile accesses to a raw pointer.

Returns a new WriteVolatile that allows volatile write-only access to ptr.

Safety

The caller must guarantee that ptr points to a value of type T that is valid for the 'static lifetime. Furthermore, ptr must be the only accessible mutable reference to the value. This is equivalent to casting the *mut T to &'static mut T.
