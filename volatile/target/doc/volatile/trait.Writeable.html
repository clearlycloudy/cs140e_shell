Trait implemented by writeable volatile wrappers.

Required Methods

unsafe fn inner(&mut self) -> *mut T

Returns the inner pointer. This function is unsafe because this returns a copy of the mutable pointer, resulting in a mutable alias.

Provided Methods

fn write(&mut self, val: T)

Writes the value val to the inner address of self. The write is always done using volatile semantics.
