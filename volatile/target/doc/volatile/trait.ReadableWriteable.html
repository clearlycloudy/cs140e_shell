Trait implemented by readable and writeable volatile wrappers.

Provided Methods

fn and_mask(&mut self, mask: T)

Applies the mask mask using & to the value referred to by self. This is equivalent to self.write(self.read() & mask).

fn or_mask(&mut self, mask: T)

Applies the mask mask using | to the value referred to by self. This is equivalent to self.write(self.read() | mask).
